#### Round
Round with 2 decimals
```
ROUND(2.3333, 2)
```

Round to the nearest integer / for .5 return EVEN integer
```
ROUND(2.5) -- Return 2
ROUND(1.5) -- Return 1
```

Round UP to the nearest integer
<b>ceiling() is the same as ceil()</b>
```
CEIL(1.5)  -- Returns 2
CEIL(1.2)  -- Returns 2
CEIL(5)    -- Returns 5 (already an integer, so no change)
```

Round DOWN to the nearest integer
```
FLOOR(1.5)  -- Returns 1
FLOOR(1.2)  -- Returns 1
FLOOR(5)    -- Returns 5 (already an integer, so no change)
```

#### Power
```
POW(4,2) -- 4^2
```
